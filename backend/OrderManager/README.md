# Order Manager

## Задание
Создание консольного приложения для формирования заказа.

### Описание
Необходимо создать консольное приложение, которое позволяет пользователю оформить заказ на товар. Приложение должно последовательно запрпшивать у пользователя информацию о заказе, а заем предоставить ему возможность подтвердить введеные данные. В случае подтверждения, пользователю выводится сообщение об успешном оформлении заказа и ожидаемой дате доставки.

### Требования
1. **Запрос данных у пользователя**:
   - Название товара
   - Количество товара
   - Имя пользователя
   - Адрес доставки
2. **Подтверждение заказа**:
   - На экране должно быть выведено сообщение с вопросом о правильности введенных данных.
   - Пример сообщения:
        `Здравствуйте, {name}, вы заказали {count} {product} на адрес {address}, все верно?`
   - Если пользователь подтверждает правильность данных, выводится сообщение об успешном оформлении заказа.
3. **Сообщение об успешном офромлении заказа**:
   - Пример сообщения:
        `{name}! Ваш заказ {product} в количестве {count} оформлен! Ожидайте доставку по адресу {address} к {todays_date + 3_days}`

### Технические требования
- Использовать editorconfig
- Проект должен быть отформатирован соглано editorconfig
- Разделять код на методы
