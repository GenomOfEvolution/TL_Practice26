# Конвертер валют

Клиентское React-приложение для отображения курсов валют.

## Установка
```bash
npm install
```

## Тесты

Покрыты сценарии:

- Рендер — все поля ввода, селекты валют, кнопки Swap и More About отображаются
- Пересчёт при изменении суммы — result обновляется после ввода нового amount
- Пересчёт при смене пары — result обновляется после выбора другой валюты
- Запрет одинаковых валют — при совпадении from/to одна из валют авто-заменяется
- Key-reset — при смене пары секция More About закрывается (сброс isOpen через key):
  `key={\`${from.code}-${to.code}\`}` — при смене валютной пары React удаляет старый
  MoreAboutSection и создаёт новый, сбрасывая isOpen в false.
  Без key пришлось бы использовать useEffect + setIsOpen(false),
  что не гарантирует сброс всех видов локального состояния (ref и т.д.).
- More About toggle — открытие/закрытие описаний валют по клику на кнопку
